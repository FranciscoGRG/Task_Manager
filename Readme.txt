Proyecto gestion de tareas monorepo basado en microservicios para implementar AWS (No se puede por ser Java 25 y no haber imagenes docker que lo soporten).

Stack para Backend:
  -Spring Boot  
  -Spring Security
  -PostgreSql
  -JPA/Hibernate
  -MinIO para almacenamiento de archivos
  -Docker
  -JWT
  -Rest Template

Stack para Frontend:
  -Angular
  -TailwindCss
